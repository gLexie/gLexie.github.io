vue-cli+webpack项目，手机无法访问本地服务

需求

在pc端启动服务（使用npm run dev），用手机访问查看页面。

问题

使用npm run dev启动服务后，pc端可以通过 http://loaclhost:8888 来访问页面（其实也无法使用本机ip:port的地址去访问页面），手机与pc连接同一无线网络，也关闭了pc端防火墙，pc端也能ping通手机的ip，手机端却始终无法通过ip:port的地址去访问页面，会显示无法访问的字样。

解决

文件package.json中，在dev命令的最后加上–host 0.0.0.0：

“scripts”: {
“dev”: “webpack-dev-server –inline –progress –config build/webpack.dev.conf.js –host 0.0.0.0”
}

重新使用npm run dev启动服务即可用手机访问本地服务提供的页面了。
